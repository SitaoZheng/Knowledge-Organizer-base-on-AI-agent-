PYTHON TUPLES: BASIC GUIDE

==========================================

1. What is a Tuple?

   - An immutable, ordered data structure in Python.

   - Defined with parentheses (), elements separated by commas.

   - Stores mixed data types; similar to lists but unchangeable.



2. Creation Methods

   # 1. Basic creation

   fruits = ("apple", "banana", "cherry")

   # 2. Single element (must add comma)

   single_fruit = ("apple",)

   # 3. Convert from list

   num_list = [1, 2, 3]

   num_tuple = tuple(num_list)  # (1, 2, 3)



3. Key Operations

   # Access element (via index, 0-start)

   print(fruits[0])  # Output: apple

   # Slice elements

   print(fruits[1:3])  # Output: ("banana", "cherry")

   # Get length

   print(len(fruits))  # Output: 3



4. Why Use Tuples?

   - Immutable: Protects data from accidental modification.

   - Faster than lists for reading operations.

   - Can be used as keys in dictionaries (lists cannot).
